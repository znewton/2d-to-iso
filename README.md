# 2D to Isometric

Convert a 2D image asset into an isometric perspective image asset.

## Prerequisites

- [Node.js](https://nodejs.org)
- [GraphicsMagick](https://www.graphicsmagick.org)

## Install

```shell
npm install
```

## Run

```shell
node index.mjs [path/to/image] [path/to/output/image]
```
